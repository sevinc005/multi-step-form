Bu layihə React.js istifadə edilərək yaradılmış çoxmərhələli bir formadır. Layihənin əsas məqsədi istifadəçi təcrübəsini (UX) artırmaq və məlumatları addım-addım toplamaqdır.

✨ Özəlliklər
Çoxmərhələli struktur: Məlumatlar 3 addımda toplanır.

Şərti Məntiq (Conditional Logic): İstifadəçi "Tələbə" seçərsə Universitet, "Professional" seçərsə Şirkət adı tələb edən sahə açılır.

Məlumatların Önizləməsi: Son addımda istifadəçi daxil etdiyi bütün məlumatları yoxlaya bilir.

Responsive Dizayn: Həm mobil, həm də masaüstü cihazlar üçün uyğunlaşdırılmış CSS dizaynı.

🚀 Texnologiyalar
React.js (Hooks: useState)

CSS3 (Flexbox, Transitions, Responsive design)

🛠️ Quraşdırılma
Layihəni lokalda işlətmək üçün:

git clone https://github.com/sevinc005/multi-step-form.git

npm install

npm start# 
Multi-Step-Form with React "React vasitəsilə hazırlanmış, şərti məntiqə (conditional logic) malik, 3 mərhələli qeydiyyat forması. İstifadəçinin statusuna (Tələbə/Professional) uyğun olaraq dinamik sahələr göstərilir."
